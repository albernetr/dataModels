# SensorModel

## Description

This entity captures the static properties of a Sensor component. 

## Example of use

```
{
  "id": "mySensor-sensor-345",
  "type": "SensorModel",
  "category": "motion",
  "controlledProperty": "acceleration",
  "function": ["sensing"],
  "brandName": "mySensor",
  "modelName": "S4Container 345",
  "manufacturerName": "mySensor Inc.",
  "dateCreated": "2014-09-11",
}
```
